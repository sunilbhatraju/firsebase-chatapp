# firsebase-chatapp
Angular Fire base chat app
install node modules using npm install

This project created with below versions
  
  npm: 5.6.0 
  Angular CLI: 1.6.7
  Node: 8.9.4
  OS: linux x64
  "angularfire2": "^4.0.0-rc.1",
  "firebase": "^3.9.0"
  
  After installing node_modules hit ng serve to run the project
